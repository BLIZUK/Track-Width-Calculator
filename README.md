# Track Width Calculator 

Приложение для инженерных расчетов ширины дорожек (Track Width) с удобным графическим интерфейсом. Разработано на **C# (WPF)** с применением архитектурного паттерна **MVVM**.

##  Технологический стек
*   **Язык:** C# 10.0
*   **Платформа:** .NET 6 / WPF
*   **Архитектура:** MVVM (Model-View-ViewModel)
*   **UI:** XAML, Data Binding, Custom Controls

##  Ключевые возможности
*   ✅ **Расчет в реальном времени:** Мгновенный пересчет параметров при изменении входных данных.
*   ✅ **Валидация данных:** Защита от ввода некорректных физических величин (например, отрицательной толщины).
*   ✅ **MVVM Подход:** Чистое разделение логики расчетов и пользовательского интерфейса.

##  Чему я научился в этом проекте
*   Реализации интерфейса `INotifyPropertyChanged` для реактивного UI.
*   Созданию переиспользуемых стилей и шаблонов (ControlTemplates) в XAML.
*   Проектированию архитектуры слабосвязанных компонентов.

##  Как запустить
1. Клонируйте репозиторий.
2. Откройте `NSA.sln` в Visual Studio.
3. Нажмите `F5`.
